## Assignment Overview

This lab assignment focuses on building a **Student Profile Web Page** using **basic HTML and external CSS**.  
The main objective is to understand **semantic HTML structure** and apply **CSS selectors** including:

- Element selectors
- Class selectors
- ID selectors
- Group selectors
- Descendant selectors
- Universal selector

Advanced CSS features and JavaScript are strictly avoided as per the assignment requirements.
